Crypto Dashboard
Een finance dashboard gemaakt met React. Je kan live crypto prijzen zien, coins zoeken, favorieten opslaan en op een coin klikken voor meer details.
Wat heeft de app?

Live prijzen van Bitcoin, Ethereum en Solana
Lijndiagram per coin
Taartdiagram van de top 10 coins op marktwaarde
Lijst van de top 100 coins
Zoekfunctie
Favorieten
Detailpagina per coin

Installatie
Zorg dat je Node.js hebt geinstalleerd. Die kan je downloaden op https://nodejs.org
Clone het project of download de bestanden en open de map in je terminal.
Installeer de packages:
npm install
Opstarten
npm run dev
De app opent op http://localhost:5173
Gebruikte technieken

React
Chart.js
CoinGecko API
